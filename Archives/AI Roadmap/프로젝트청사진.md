## ✅ Phase별 세부 내용

### Phase 1: 데이터 인프라
- 데이터 핸들러 (`data_handler.py`) 구축
- CSV 캐싱, 검증, 결측치 경고
- yfinance → pykrx 폴백 로직
- 데이터 품질 리포트 자동 생성

### Phase 2: 전략 구현
- 기본 전략: SMA Cross
- 확장 전략: CANSLIM, DoubleBottom, Envelope 등
- JSON 기반 전략 정의 스키마 설계

### Phase 3: 백테스트 엔진
- 단일 전략 실행 엔진(backtester)
- 병렬 실행기(parallel_runner)로 멀티전략/종목 처리
- 테스트: 성능/부하 검증, 통합 테스트

### Phase 4: 리포팅 & 시각화
- 실행 결과 리포트 (CSV, HTML, Chart)
- 전략별/종합 성과 분석
- 리스크 지표, 트레이드 로그, 그래프

### Phase 5: 아카이빙 & 운영
- 레거시 코드/자료 관리
- 패키지 의존성/환경 고정
- CI/CD, 자동화 파이프라인
- 장기 운영 고려(확장성/해외주식 등)
